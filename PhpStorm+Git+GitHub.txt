

1. Setting - Version Control - Git
2. Теперь в вкладке VCS включить интеграцию с системой контроля версий - VCS->Enable Version Control Integration...: выбираем Git и жмем Ok. Файлы станут красные.
3. Перед началом инициализации желательно настроить игнорируемые файлы в папке .gitignore в корне проекта.
4. Пр.кн.мы. на папке проекта - Git - +Add. Файлы станут зеленые.
5. Расположить проект в репозиторий GitHab. VCS - Import into Version Control - Share Project on GitHab. 
6. Имя репозитория, подробности о проекте - Share. Отправит файлы в репозиторий.

Если сделать изменения, то можно отправить в репозиториЙ.
1. Кнопка с зеленой стрелкой VCS - указать сообщение - Commit. Сделать VCS - Git - Push - Push.
